#README

A Python game where you choose a story and fill in the blanks with your own words to create a story.

python main.py How to play

The program will show you 3 story templates to choose from:

1.Hospital Stay

2.Camping Trip

3.Enchanted Castle

Type the number of the story you want (1, 2 or 3) and press Enter! The program will ask you to type different kinds of words one by one (like a noun, a color, an animal, etc.) After you enter all the words, your story will appear on the screen!

Example Welcome to Mad Libs! Pick a story template: 1 - Hospital Stay 2 - Camping Trip 3 - Enchanted Castle Enter 1, 2 or 3: 1

Type a number: 5

Type a measure of time: weeks

Type a mode of transportation: skateboard
